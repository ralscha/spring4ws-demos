Spring Framework 4 / SockJS / AutobahnJS Demos

Demos

  • Chat

    This is the sample application from portal-java.

  • Bryntum Ext Scheduler / Bryntum Touch Scheduler

    This is a port from a Ext JS/Sencha Touch/Node.js sample application published on this blog: http://www.bryntum.com/blog/under-the-hood-of-the-socket-io-express-example/.

  • Twitter

    Twitter demo that uses the Streaming API from Twitter and tracks these terms "ExtJS", "#extjs", "#SenchaTouch", "Sencha", "#java", "java7", "java8", "#websocket", "#SpringFramework", "html5" and "javascript".

  • Snake

    Port of the Websocket snake demo included in the Tomcat 7 distribution (webapps/examples folder).

  • Ext JS Grid

    Demonstrates an Ext JS store and model with a WAMP proxy. Create, update and delete operations are broadcasted to all connected clients.

  • Access Log Wargames Map

    Translates ip addresses from the Apache access log from this server to latitude/longitude with the help of GeoCityIP and visualizes them on a world map. This example is based on maptail and mape's wargames.

  • Smoothie Charts

    Copy of the Smoothie Charts Server Load Example. But in this example the server generates the random data every second and sends it to the connected clients.

  • Google Maps Demo

    Server sends periodically coordinates to the connected clients that moves two car icons on a google maps.

  • Portfolio

    Copy of the traditional portfolio demo from Rossen Stoyanchev's rstoyanchev/spring-websocket-portfolio project

  • Realtime Bandwidth Monitoring

    Shows received and sent bytes of this server network interface. Port of the pesarkhobeee/Realtime-Bandwidth-Grapher project.

  • Drawboard Example

    Port of the drawboard example from Tomcat 8 RC5. Websocket only example.
